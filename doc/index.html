A Lightweight UI Framework

tui.js 什么是

tui.js 是一个轻量级的javascript UI 框架， 目标是让开发人员用尽可能少的代码完成尽可能多的任务(Write Less Do More)， 当然这是一个流行的口号，每个框架都会尽可能的按照这个原则来实现，不过手段各有不同。 对比 bootstrap，tui.js 使用的 html 标签更少，内置了更多的控件， tui.js 更加侧重于帮助程序员处理日常常用的操作，如：数据的收集、处理、提交和展现等。 而 bootstrap 则只是提供了一个 css 框架，很少关心与数据相关的操作。 为了方便的页面设计，tui.js 尽量保证一个控件只使用一个标签， 这样就避免了记忆大量标签组合带来的烦恼，和 bootstrap 相比可以节省大量 html 代码。 不过 bootstrap 同时还专门为响应式布局做了优化，这点目前 tui.js 并没有过多涉及， 如果想要开发支持响应式布局的应用，需要配合 css3 媒体查询来自己实现。 总之 tui.js 比较适合有大量数据交互，如企业ERP应用等场合。

开始使用吧

前台多语言支持

为了支持多语言页面展示，往往需要后台代码提供语言资源文件， 并使用 jsp 或 asp 等类似的页面脚本来进行开发。 jsp 等技术的问题是无法做到动静分离，往往为了支持一个很小的动态特性， 就要把原本完全可以用静态页展示的页面也做成了 jsp 浪费了服务器处理时间， 在高并发的环境下也就需要更多的web服务器来进行处理... 
var dict = {
    "This is a test": "这是一个测试",
    ...
};
tui.registerTranslator("zh-cn", dict);
...
tui.msgbox(str("This is a test"));

灵活的数据表格

数据处理最常用的就是表格控件，tui.js 提供的数据表格支持整体加载百万记录大数据集、 使用虚拟滚动可以支持上亿条记录、拖拽修改列宽、首行固定、自动滚动条、横向滚动条、JSON 数据绑定、 排序、自定义 formatter、行选择、行内编辑、键盘控制等...

AJAX 数据表单

现代 web 应用的前后台数据表交互更多的依赖 AJAX 来完成以减少页面的刷新， 加强友好性以及提高效率。大量使用 AJAX 脚本不但不方便，且由于缺少固定模式， 代码量过多，尤其是都是异步回调事件的处理，使代码不容易看懂。tui.js 非常注意前后台交互， 提供了专门的数据表单控件，用来代替js脚本，不但代码量小，逻辑更清楚，且功能强大。 tui 表单可以从其他 tui 控件中组合出各种类型的json数据形式，支持自动数据验证。 支持数据回显，使得数据的操作非常简单...
<span class="tui-form" data-action="api/query" data-method="get" >
    <span data-field="name"
    data-target="aTextInput"></span>
    <span data-field="sex"
    data-target="aSelecInput"></span>
</span>
<span class="tui-input" data-type="select"></span>
<a class="tui-button" data-type="select">按钮</a>
<div class="tui-calendar" data-type="select"></div>
...

内置丰富的控件

tui.js 提供了许多内置的控件，如：按钮、单选框、复选框、 日历、对话框、文本框、选择框、日历选择框、文件上传框、自定义输入框、自定义选择框、 普通表格、数据表格、列表框（包括树）、弹出层、弹出菜单、滚动条、页签、数据表单、 分页器、导航条、折叠框以及折叠框组等...

基于 TypeScript 很好扩展

tui.js 使用 TypeScript 和 less 编写而成， 由于 TypeScript 良好的结构化和面向对象特性，使得代码逻辑更加清晰易懂， 也更方便扩展控件，想实现一个没有提供的控件，可以根据类似的控件继承再修改， 也可以直接继承自 tui.ctrl.Control 基类，完全自己定制 ...
export class DemoCtrl extends Control<DemoCtrl> {
    static CLASS: string = "your-class-name";
    constructor(el?: HTMLElement) {
        // ......
    }
}

夸浏览器支持

tui.js 兼容世面上常见的各种浏览器，IE最低支持到IE8，如果有需要也可以支持到IE7。
